# Credit Card Transaction Analysis

## Overview
Analyzed credit card transaction data using MySQL to solve business and analytical problems.

## Questions Solved

1. Top 5 cities by spend
2. Highest spend month by card type
3. Cumulative spend milestone analysis
4. Gold card spend percentage
5. Highest and lowest expense type by city
6. Female spend contribution analysis
7. Month-over-month growth analysis
8. Weekend spending behavior
9. 500th transaction analysis

## SQL Skills Used

- CTEs
- Window Functions
- ROW_NUMBER()
- LAG()
- Aggregate Functions
- Date Functions
- Conditional Aggregation
