# SQL_Projects
This repository contains SQL projects and case studies focused on solving real-world business problems using MySQL.

## Skills Demonstrated

- SQL Querying
- Common Table Expressions (CTEs)
- Window Functions
- Aggregations
- Ranking Functions
- Time-Series Analysis
- Business Analytics
- KPI Reporting

## Projects

### Credit Card Transaction Analysis

Analyzed credit card transaction data to answer business questions such as:

- Top spending cities and contribution analysis
- Highest spending month by card type
- Cumulative spend milestones
- Gold card spend analysis
- Highest and lowest expense categories by city
- Female spending contribution by expense type
- Month-over-month growth analysis
- Weekend spending behavior
- Transaction velocity analysis

### Technologies

- MySQL
- SQL Window Functions
- Analytical Query Design
